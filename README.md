# Azure VM Web Hosting Project 🚀

## 📌 Overview
This project demonstrates hosting a website on an Azure Virtual Machine.

## 🛠️ Steps
1. Created Azure VM (Linux)
2. Connected using SSH
3. Installed Nginx
4. Configured NSG (Port 80)
5. Hosted HTML webpage

## 🌐 Output
Website is live using public IP
